# Metric-Imperial Converter
The purpose of this project is to create unit tests and functional tests in Express.js with Chai library.

### Supports the below convertions:

* gallons to Liters.
* Liters to gallons.
* miles to kilometers.
* kilometers to miles.
* libras to kilograms.
* kilograms to libras.
